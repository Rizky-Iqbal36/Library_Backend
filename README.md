## Installation

Note: <br />

- Make sure you have docker installed on your computer if you don't, please feel free to access [Docker Documentation](https://docs.docker.com/)
- Or you can install the service i use on docker compose instead install it using docker

```bash
$ cp .env.example .env

$ npm install
```

## Running the app

```bash
# Start service
$ npm run dependency:local

# Development
$ npm run start

# Watch mode
$ npm run start:dev

# Production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
